The Manifesto

Somewhere in a climate-controlled office park, a surfacing specialist is massaging a Class-A body panel to G2 curvature continuity, zooming to 4000% to chase a highlight across a fender that will never be looked at that closely by anyone, ever, including God.

Meanwhile, down here in the swamp, we know the truth: every NURBS surface is a liar. An infinitely smooth parametric fantasy propped up by control points, knot vectors, and the collective denial of the CAD-industrial complex. Your GPU never rendered a NURBS in its life. The moment anything real happens — a render, a toolpath, a 3D print, a finite element daring to be analyzed — that pristine mathematics gets tessellated into triangles anyway, quietly, behind your back, like a shameful family secret.

The deNURBurator simply does in public what your kernel does in private. It drags the B-rep out of the STEP file, beats it flat into honest polygons, and displays the remains — rotating, lit, and z-buffered — in the only rendering target that never lied to anyone: the terminal.

Thirteen ASCII characters. That is the entire shading gamut you get:  .,-~:;=!*#$@ — from the void to full specular ecstasy. A fixed light vector glares down from the upper left like a porch lamp in the fog. Face normals are dot-producted against it without anesthesia. There is a real perspective camera, a real depth buffer, and real rotation matrices, all hand-rolled in numpy, because a rendering pipeline you can read in one sitting is worth ten black-box engines.

Is this useful? Aggressively. You can sanity-check a vendor's STEP file over SSH, on a headless box, in the sixty seconds before a meeting, without launching a CAD package that needs eleven gigabytes and a license server's blessing. And when the part appears — chunky, glowing, spinning in green phosphor like it's 1994 and the swamp modem just handshook — you will feel something no fillet dialog ever gave you.

Mathematical perfection is a rental. Triangles are forever.

Seize the Machine

One script. No framework. No telemetry. No account. Copy, paste, desecrate:

░ INSTALLATION RITUAL ░ 
# summon dependencies (the swamp provides)
python -m pip install numpy trimesh cascadio

# windows folk also require the ritual bindings
python -m pip install windows-curses

# seize the machine
curl -fsSLO https://peterjamesvilla.com/denurburator/denurburator.py

# feed it your most precious parametric surfaces
python denurburator.py YOUR_PART.step

# no STEP file handy? violate the built-in nitinol actuator
python denurburator.py --demo

Prefer the heavyweight OpenCascade experience? pip install cadquery works as a fallback backend and tessellates STEP natively. The deNURBurator tries trimesh + cascadio first, then cadquery, then insults you with a helpful error message.

Technical Autopsy

OrganWhat It Does To Your Geometry
TESSELLATOR trimesh + cascadio (or cadquery) parses the STEP B-rep and force-flattens every NURBS patch into raw triangles. Chunkiness is adjustable with --tol. Bigger tolerance = more violence.
DECIMATOR Assemblies over the face budget (--max-faces, default 1800) get quadric-decimated if fast-simplification is present — or randomly cleaved if it is not. Either way they fit.
PROJECTOR Hand-rolled numpy: X/Y rotation matrices, perspective divide against a camera parked at --distance part-radii, and a 2× horizontal stretch because terminal cells are tall and nobody can stop them.
Z-BUFFER Per-character depth buffer with barycentric interpolation, filled front-to-back so hidden fragments die cheap. Your occluded faces stay occluded, as nature intended.
ILLUMINATOR One fixed light vector, dot-producted against every face normal, mapped onto the sacred ramp  .,-~:;=!*#$@. Feral STEP winding orders are handled by flipping normals camera-ward. Flat shading. Gouraud is for cowards.
THE LOOP curses, flicker-free, targeting --fps 30. Pause, zoom, and spin-rate keys included. --scale handles the spread from warehouse-sized assemblies (0.5) to a lone nitinol actuator (1.5+). --dump N prints frames straight to stdout for piping, CI, or framing.
