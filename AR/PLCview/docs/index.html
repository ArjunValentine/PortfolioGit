What PLCview is

PLCview is a small BlueVector demo that turns a PLC-to-AR-glasses backplane diagnostics architecture into something concrete: pick a chassis, a data-access method, a gateway placement, and a glasses platform in the Configurator, and it draws the information-flow diagram live and generates a downloadable MIT-licensed starter code package wired to those choices.

This page holds a static reference copy of that diagram — useful for a deck or a doc — plus the project's source.
